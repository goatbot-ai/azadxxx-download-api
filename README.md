# AutoDownload Bot + API

## Deploy API
1. `npm install`
2. `node index.js`
3. API URL: `https://your-api-url.com/api/download?url=<video_link>`

## Deploy Bot
1. Replace API_BASE in autodownload.js
2. Deploy bot
3. Supports:
   - `/autodownload <link>` command
   - Auto-download on group link share
